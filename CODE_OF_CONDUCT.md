# Quy tắc ứng xử

Dự án áp dụng [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Tóm tắt: tôn trọng nhau, không quấy rối/phân biệt, phản hồi kỹ thuật nhắm vào code không nhắm
vào người. Báo cáo vi phạm: mở issue confidential hoặc email maintainer (bổ sung khi lập org).
