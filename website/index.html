OpenITMS-SMB

Nền tảng mã nguồn mở quản lý hạ tầng IT cho doanh nghiệp SMB

Đang phát triển — chưa có bản phát hành chính thức

Tự động hóa triển khai, chạy lệnh/script xuống Windows 11 và Linux từ một Web UI. Cài trong 10 phút, 1 lệnh, zero-config — không cần biết Docker hay Ansible.

Core sạch

Fork Semaphore UI (MIT), sửa core tối thiểu qua patch script-hóa — sync bản vá bảo mật ≤ 48h.

Plugin-first

Mở rộng qua HashiCorp go-plugin (gRPC). Viết plugin bằng Go hoặc Python. Registry ký số.

Zero-config

Toàn binary native (core + MariaDB + PowerShell) — tuyệt đối không Docker. Ném cert vào là chạy.

AI-driven dev

Nạp yêu cầu qua GitHub Issues → AI tự đọc, tự phát triển → quality gate → người duyệt.
